# Security Policy

## Supported Versions

Only the latest stable release of `maatify/infra-drivers` is supported with security updates.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do NOT open a public issue.**

Instead, contact:

- Email: security@maatify.dev

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Potential impact

We aim to acknowledge reports within **72 hours**.

---

## Scope

This policy applies only to the `maatify/infra-drivers` package.

Issues originating from:
- application code
- bootstrap logic
- environment configuration
- adapter layers

are considered **out of scope**.
